# Les Goûts Délicieux

Site web moderne et professionnel pour un restaurant, avec page d’accueil, menu dynamique, réservation, panier et panneau d’administration.

## Structure
- css/ : styles du site
- js/ : interactions dynamiques
- backend/ : scripts PHP pour le menu, réservations, commandes et administration
- sql/ : script SQL de création de la base de données
- images/ : ressources visuelles

## Installation
1. Créez une base MySQL et exécutez le script SQL dans [sql/database.sql](sql/database.sql).
2. Configurez l’accès MySQL dans [backend/config.php](backend/config.php).
3. Placez le projet dans le dossier racine de votre serveur local (par exemple htdocs pour XAMPP/WAMP).
4. Ouvrez l’URL du projet dans votre navigateur.

## Fonctionnalités
- Page d’accueil moderne
- Menu dynamique affiché depuis la base de données
- Système de réservation enregistré en base
- Panier et validation de commande
- Tableau d’administration pour gérer les plats, réservations et commandes
